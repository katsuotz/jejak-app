# Jejak

A monorepo for **Jejak**, a mobile-first PWA to help runners improve their cadence.

## Structure

```
├── frontend/    # SvelteKit PWA (Tailwind CSS + shadcn-svelte)
└── ...          # Future services (backend, etc.)
```

## Getting Started

```bash
cd frontend
pnpm install
pnpm dev
```

See [frontend/README.md](./frontend/README.md) for full details on features, tech stack, and project structure.
